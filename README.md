# Marvel Order

An Open-source Interactive version of the upcoming Marvel Films and Series ordered chronologically and by release so you
can see the latest correct dates.

I've been wanting a way to keep track of all the Marvel movies and shows so since there's so much going on and they
frequently swap positions for different releases.

I based it on these types of images so you can see how the order is updated over time.
https://www.reddit.com/r/weeklyplanetpodcast/comments/kb3ibw/updated_marvel_release_schedule/
